# Features

## Public Site

- Trang giới thiệu, kinh nghiệm, dự án, mục tiêu, liên hệ.
- Responsive layout.
- Form liên hệ lưu vào Supabase.

## Admin Panel

- Đăng nhập OAuth (Google/GitHub).
- Quản lý nội dung: giới thiệu, kinh nghiệm, dự án, mục tiêu.
- Quản lý kho ảnh.

## Media

- Upload ảnh lên Cloudinary.
- Lưu `public_id` để xóa ảnh trên Cloudinary khi xóa khỏi kho.
